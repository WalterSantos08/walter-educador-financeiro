# Exemplos e Referências

Esta pasta contém exemplos e materiais de apoio utilizados no desenvolvimento do projeto **walter-educador-financeiro**, um assistente virtual financeiro com IA.

O objetivo é demonstrar, de forma prática, como cada etapa do projeto foi construída, desde a definição do problema até a implementação do chatbot funcional.

## 📂 Estrutura de Exemplo

Na pasta `src/` você encontrará um exemplo de aplicação funcional utilizando:

- Streamlit para interface interativa  
- Integração com modelos locais via Ollama  
- Dados simulados para personalização das respostas  
- Contexto dinâmico baseado no perfil do usuário  

---

## 💡 Objetivo desta pasta

Esta seção serve como apoio para:

- Entender a construção do assistente passo a passo  
- Explorar boas práticas no uso de IA generativa  
- Aprender como estruturar projetos com dados e chatbot  
- Facilitar futuras melhorias e expansões do projeto  

---

## 🚀 Sobre o projeto

O **walter-educador-financeiro** é um assistente virtual que:

- responde dúvidas sobre finanças pessoais  
- analisa padrões de gastos  
- explica conceitos financeiros de forma simples  
- utiliza IA para gerar respostas contextualizadas  
- prioriza educação financeira, não recomendação de investimento
